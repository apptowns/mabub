# 중요 설정
## 보상
### 1. 골드
1) 개념
    - 게임의 기본 재화 
2) 보상 
    - 전투를 통해 확보되는 재화 
    - 전투외 확보처
      - 골드 던전
      - 인게임에서 광고 보상 상자
      - 방치 보상 보너스
3) 사용처
    - 강화에 사용되는 기본 재화 

### 2. 다이아
1) 개념
    - 골드외에 부개 재화 
    - 유료 재화로 샵에서 구매 가능하다.
2) 보상
    - 우편함으로 보상 지급
    - 퀘스트를 통해 보상
    - 이벤트를 통해 보상 
    - 인게임 광고 보상
    - 보스 던전을 통해 보상 
3) 사용처
    - 샵에서 주로 장비, 동료, 스키 소환에서 사용
    - 로봇 구매시에 사용 
    - 각종 컨텐츠의 즉시 완료시 사용
    - 상점 리셋등 리셋할때 사용
### 3. 룬
1) 개념
    - 로보트, 마물 모두와 연계되어 있는 미치의 물체
    - 파랑, 빨강, 녹색, 노랑, 보라 다섯가지 색갈의 룬이 있다.
    - 파랑 : 물의 룬 (물)
    - 빨강 : 불의 룬 (화염)
    - 녹색 : 땅의 룬 (랜드)
    - 노랑 : 빛의 룬 (라이트닝)
    - 보라 : 어둠의 룬 (흑관련)
2) 보상
    - 아지트 던전 파괴시에 보상 (랜덤하게 나온다.)
    - 광산에서 보상 
    - 추가 기획으로 룬 확보 
3) 사용처
    - 로봇 강화 로봇 / 로봇 선택 / 강화    
    - 마제라티 : 특성을 찍는 곳에 필요 
    - 특성 : 룬의 경향성 랜덤하게 분류할수 있음_
